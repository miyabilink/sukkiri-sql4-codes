## スッキリわかる入門シリーズとは
「初学者ファースト」を旗印に、Java・Python・C言語・SQLなどをラインナップするIT入門書シリーズです。現役の「技術者 兼 技術指導者」のみが著者として執筆し、初学者特有の悩みやつまづきに対してきめ細かくフォローアップできる構成・解説を心がけています。

　→ [執筆著者陣によるサポートサイト (sukkiri.jp）](https://sukkiri.jp/)
